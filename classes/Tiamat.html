open_local (num_parallel, *requires, &block)

Launch num_parallel local Ruby interpreters.

If a block is given:

  • Pure.worker is replaced for the duration of the block.

  • The return value is the block result.

  • The interpreters are stopped when the block finishes.

If no block is given, an opened worker is returned. You are responsible for closing it.
